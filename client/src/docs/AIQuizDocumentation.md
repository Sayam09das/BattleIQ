# AI Quiz Generator Documentation

## Description
The AI Quiz Generator allows educators and learners to automatically create quizzes based on a topic or text. It saves time and provides personalized practice materials.

## Features
- Input a topic or text to generate multiple-choice or short-answer questions.
- Adjust difficulty: Easy, Medium, Hard.
- Export quizzes for practice or classroom use.
- Analyze performance with tracking tools.

## How to Use
1. Go to the AI Quiz Generator page.
2. Enter your topic or text in the input box.
3. Select the difficulty level.
4. Click "Generate Quiz".
5. Review generated questions.
6. Export or share as needed.

## Tips
- Provide clear and specific topics for better quiz results.
- Use quizzes for quick revisions or test preparation.
